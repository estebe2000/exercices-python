Guide d'utilisation

La page d'accueil vous présente :

  • Une vue d'ensemble des fonctionnalités de l'application
  • Des accès rapides aux différentes sections (Exercices, Cours, Administration)
  • Des informations sur l'application et son utilisation

Vous pouvez naviguer vers les différentes sections en utilisant :

  • Les cartes sur la page d'accueil
  • La barre de navigation en haut de la page

Pour générer un nouvel exercice :

  1. Accédez à l'onglet Exercices dans la barre de navigation
  2. Sélectionnez un niveau (Troisième, SNT, Prépa NSI, Première ou Terminale)
  3. Choisissez un thème parmi ceux disponibles
  4. Définissez un niveau de difficulté (de 1 à 5)
  5. Cliquez sur le bouton Générer l'exercice

L'IA va alors créer un énoncé personnalisé avec un squelette de code à compléter et des tests pour vérifier votre solution.

Mode débutant : Les exercices marqués comme "débutant" (généralement Troisième, SNT et Prépa NSI) sont conçus sans fonctions ni classes, adaptés aux débutants en programmation.

Télécharger en notebook Jupyter : Après avoir généré un exercice, vous pouvez cliquer sur le bouton Télécharger en notebook Jupyter pour obtenir l'exercice au format .ipynb, utilisable dans Jupyter Notebook ou JupyterLab.

Une fois l'exercice généré :

  1. Écrivez votre code dans l'éditeur de code
  2. Cliquez sur Exécuter le code pour voir le résultat
  3. Le résultat s'affichera dans la section Résultat de l'exécution

L'éditeur de code dispose de la coloration syntaxique et de l'indentation automatique pour faciliter l'écriture du code.

Pour obtenir une évaluation de votre code :

  1. Après avoir écrit votre solution, cliquez sur Évaluer le code
  2. L'IA analysera votre code et fournira des commentaires détaillés
  3. L'évaluation inclut :
    • Vérification de la conformité à l'énoncé
    • Identification des erreurs potentielles
    • Suggestions d'amélioration
    • Conseils pour aller plus loin

La Gestion Électronique de Documents (GED) vous permet de :

  1. Uploader des documents de différents formats
  2. Organiser vos documents avec des tags
  3. Rechercher des documents par nom ou par tag
  4. Marquer des documents comme cours pour les afficher dans l'onglet Cours

Pour ajouter un document :

  1. Cliquez sur Admin dans la barre de navigation
  2. Sélectionnez GED dans le menu déroulant
  3. Cliquez sur Ajouter un document
  4. Sélectionnez un fichier à uploader
  5. Ajoutez des tags pour organiser votre document
  6. Cliquez sur Enregistrer

Pour rechercher des documents :

  1. Utilisez la barre de recherche pour filtrer par nom
  2. Cliquez sur un tag pour filtrer les documents par tag

Pour modifier un document :

  1. Cliquez sur Modifier à côté du document
  2. Modifiez le nom et/ou les tags
  3. Cliquez sur Enregistrer

Pour marquer un document comme cours :

  1. Cliquez sur le bouton avec l'icône de livre à côté du document
  2. Le bouton devient vert plein lorsque le document est marqué comme cours
  3. Cliquez à nouveau pour retirer le document des cours

La Bibliothèque de Cours vous permet de :

  1. Consulter les documents marqués comme cours dans un style bibliothèque
  2. Filtrer les cours par tags
  3. Rechercher des cours par nom

Pour consulter vos cours :

  1. Accédez à l'onglet Cours dans la barre de navigation
  2. Les documents sont affichés sous forme de cartes avec leur nom, description et tags
  3. Utilisez la barre de recherche pour filtrer par nom
  4. Utilisez le sélecteur de tags pour filtrer par tag
  5. Cliquez sur Consulter pour ouvrir un document

Note : Seuls les documents marqués comme cours dans l'onglet GED sont visibles dans la Bibliothèque de Cours.

Vous pouvez choisir entre trois modèles d'IA :

  • LocalAI (Mistral) : Modèle hébergé localement sur votre machine, idéal pour une utilisation sans connexion internet
  • Google Gemini : Modèle en ligne de Google, offrant d'excellentes performances pour la génération d'exercices
  • Mistral (Codestral) : Modèle spécialisé pour le code via l'API Mistral, particulièrement efficace pour l'évaluation de code Python

Pour changer de modèle :

  1. Cliquez sur Admin dans la barre de navigation
  2. Sélectionnez Configuration dans le menu déroulant
  3. Sélectionnez le modèle souhaité
  4. Cliquez sur Enregistrer

Note : Le modèle sélectionné est affiché en bas de page. Pour utiliser Gemini ou Mistral, vous devez configurer les clés API correspondantes dans le fichier .env.

L'application vous permet de télécharger les exercices générés au format notebook Jupyter (.ipynb) :

  1. Après avoir généré un exercice, cliquez sur Télécharger en notebook Jupyter
  2. Donnez un titre à votre notebook
  3. Cliquez sur Télécharger
  4. Le notebook sera téléchargé sur votre ordinateur

Les notebooks générés contiennent :

  • L'énoncé de l'exercice formaté en Markdown
  • Le squelette de code à compléter
  • Les tests pour vérifier votre solution

Vous pouvez ouvrir ces notebooks avec Jupyter Notebook, JupyterLab, Google Colab, ou tout autre environnement compatible avec le format .ipynb.

Le menu Admin donne accès à plusieurs fonctions d'administration :

  • GED : Gestion Électronique de Documents pour uploader, organiser et gérer vos documents
  • Éditeur d'exercices : Interface pour modifier le fichier data.json qui contient les définitions des exercices
  • Configuration : Paramètres de l'application, notamment le choix du modèle d'IA

L'éditeur d'exercices vous permet de :

  1. Ajouter de nouveaux niveaux scolaires
  2. Ajouter de nouveaux thèmes à un niveau
  3. Ajouter de nouveaux exercices avec différents niveaux de difficulté
  4. Marquer des exercices comme "débutant" (sans fonctions ni classes)
  5. Modifier les exercices existants
