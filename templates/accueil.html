À propos de cette application

Cette application a été conçue pour faciliter l'apprentissage et l'enseignement du Python dans un contexte scolaire. Elle offre plusieurs fonctionnalités :

  • Génération d'exercices Python personnalisés adaptés à différents niveaux scolaires
  • Exécution et évaluation de code Python directement dans l'application
  • Gestion électronique de documents (GED) pour organiser vos ressources
  • Bibliothèque de cours en style visuel pour faciliter l'accès aux documents
  • Utilisation de l'intelligence artificielle pour générer et évaluer les exercices
